# Game of Life/生命游戏  
## 一、生命游戏规则简介：  
每个细胞有两种状态——存活或死亡，每个细胞与以自身为中心的周围八格细胞产生互动。  
当前细胞为死亡状态时，当周围有3个存活细胞时，该细胞变成存活状态。  
当前细胞为存活状态时，当周围有2个或3个存活细胞时， 该细胞保持原样。  
其他情况下，细胞均为死亡状态。   
## 二、基本功能：  
1.简单的图形化界面  
2.自定义棋盘大小  
3.自定义初始生成的活细胞的数量  
4.预设了几种基础图形  
5.允许游戏中实时增删活细胞
